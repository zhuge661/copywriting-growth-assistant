# Creator Profile

This file defines who the skill serves, what topics it writes about, what voice it uses, and which platforms it optimizes for.

Read this file before every collaboration. The open-source version does not include any private creator profile, samples, fixed ending, slogan, or dedicated content lanes.

---

## Current Default Configuration

The open-source version does not preset a real creator.

Before first use, fill in the blank template below. Do not start writing until these fields are complete:

- Creator / IP Name
- Account Name
- Core Audience
- Default Platforms
- Content Lanes
- Fixed Spoken Ending
- Brand Hashtag
- English Fixed Ending
- English Brand Hashtag
- English Audience / Region
- Hard Taboos

## Pre-Use Check

- If the active creator is unclear, ask the user to fill in the profile first.
- If content lanes are missing, do not invent a long-term content matrix for the creator.
- If fixed endings or brand hashtags are missing, use placeholders and ask the creator to fill them in.
- If English or bilingual content is requested but English ending, English hashtag, or English audience is missing, ask for them first.
- If a dedicated profile exists, load only the active creator's profile. Do not load another creator's rules or samples as default evidence.

---

## Migration Rules

Can migrate:

- fact-check first;
- digest before drafting;
- confirm the core claim before writing;
- full / semi-fast / fast modes;
- fact / angle / risk / do-not-expand preparation format;
- bilingual analysis workflow;
- sample-based growth mechanism;
- self-check structure.

Must refill:

- creator identity, account name, audience, platforms;
- content lanes and KPIs;
- fixed endings and brand hashtags;
- English endings, English hashtags, English audience;
- voice boundary, taboos, topics to use and avoid.

Do not migrate by default:

- any creator's fixed slogan;
- any creator's content lanes;
- any creator's sample conclusions;
- any creator's dedicated expression rules.
