## Archive Rules

- Before archiving, compress old sample value into a stage rule summary.
- Archived samples preserve original text, but they are not default writing evidence.
- Default writing evidence priority:
  1. current creator profile;
  2. solidified rules;
  3. rule summary;
  4. latest 5 complete samples;
  5. archive index;
  6. full archive only when explicitly requested by the creator.
